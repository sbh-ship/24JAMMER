# Список компонентов 24JAMMER

## Основные компоненты

### Микроконтроллер
- **ESP32** — микроконтроллер с двойным ядром, поддержка Wi-Fi и Bluetooth
  - Частота: 240 MHz
  - RAM: 520 KB
  - Flash: 4 MB (минимум)

### Радиомодули
- **2x Ebyte E01-ML01DP5** (рекомендуется) или совместимые nRF24L01+
  - Частота: 2.4 ГГц
  - Мощность: 100 мВт (+20 dBm)
  - Дальность: до 500+ метров на улице
  - Стабильность: высокая благодаря встроенным PA/LNA усилителям

### Питание
- **Источник питания**: 3.3V, минимум 1A (рекомендуется 2A+ для максимальной мощности)
- **Конденсатор**: 100μF (электролитический) для каждого nRF24L01+ рядом с пином VCC
- **Конденсаторы**: 0.1μF (керамические) для развязки каждого модуля

### Компоненты управления
- **Светодиод**: RGB или обычный красный LED на GPIO 2 (индикация статуса режима)
  - Резистор: 330Ω для ограничения тока
- **Кнопка**: Тактильная кнопка на GPIO 0 (BOOT) для переключения режимов
  - Замыкание на GND (INPUT_PULLUP)

## Дополнительное оборудование
- **USB кабель**: Micro-USB для программирования ESP32
- **Провода**: Разноцветные провода 24-26 AWG для соединений
- **Паяльник**: 30-40W для монтажа компонентов
- **Макетная плата** или печатная плата для постоянной сборки

## Схема подключения

### SPI шина (общая для обоих модулей)
| Сигнал | ESP32 GPIO | Описание |
|--------|------------|----------|
| **SCK** | GPIO 14 | Тактовая частота шины SPI |
| **MOSI** | GPIO 13 | Выход данных master-шины |
| **MISO** | GPIO 12 | Вход данных master-шины |

### nRF24L01+ Модуль 1
| Сигнал | ESP32 GPIO | Описание |
|--------|------------|----------|
| **VCC** | 3.3V | Питание (с конденсатором 100μF) |
| **GND** | GND | Общая земля |
| **CE** | GPIO 16 | Chip Enable (включение передачи) |
| **CSN** | GPIO 15 | Chip Select (выбор микросхемы) |
| **SCK** | GPIO 14 | Общая SPI шина |
| **MOSI** | GPIO 13 | Общая SPI шина |
| **MISO** | GPIO 12 | Общая SPI шина |

### nRF24L01+ Модуль 2
| Сигнал | ESP32 GPIO | Описание |
|--------|------------|----------|
| **VCC** | 3.3V | Питание (с конденсатором 100μF) |
| **GND** | GND | Общая земля |
| **CE** | GPIO 18 | Chip Enable (включение передачи) |
| **CSN** | GPIO 17 | Chip Select (выбор микросхемы) |
| **SCK** | GPIO 14 | Общая SPI шина |
| **MOSI** | GPIO 13 | Общая SPI шина |
| **MISO** | GPIO 12 | Общая SPI шина |

### Управление
| Компонент | ESP32 GPIO | Описание |
|-----------|------------|----------|
| **LED (RGB/Red)** | GPIO 2 | Индикатор статуса режима |
| **Кнопка BOOT** | GPIO 0 | Переключение режимов (замыкание на GND) |

## Важные замечания по питанию

⚡ **Модули nRF24L01+ особенно чувствительны к качеству питания:**
- Используйте отдельный стабилизатор 3.3V (AMS1117 или подобный) для питания радиомодулей
- Обязательно установите конденсатор 100μF рядом с каждым модулем
- Кабели питания должны быть толстыми и короткими
- При использовании модулей PA/LNA (Ebyte E01) питание должно быть стабильным и мощным (минимум 1.5A)

⚡ **Для максимальной мощности:**
- Используйте Ebyte E01-ML01DP5 (100 мВт с усилением)
- Питание 3.3V должно быть способно выдать минимум 1.5-2A
- Дополнительная развязка 0.1μF керамическими конденсаторами между VCC и GND у каждого модуля
